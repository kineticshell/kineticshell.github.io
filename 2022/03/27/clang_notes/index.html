printf参数

变量名 数据类型 参数 大小（字节）
int 整型 %d 4
long 长整型 %ld 8
short 短整型 %h 2
unsigned 无符号整型 %u 4
char 字符 %c 1
float 浮点数 %f 4
double 双精度浮点数 %f 8

_Bool 布尔值
_Complex 复数

参数 功能
%d, %i 打印十进制整型
%o 打印八进制数据
%#o 打印前缀为0的八进制数据
%x 打印十六进制数据

%Le 打印科学计数法表示的long double类型的数据
%La 印在十六进制下用科学计数法表示的long double类型的数据
%g 自动根据数据类型选择打印的值
%zd 专用于打印sizeof和strlen的返回值
p% 指针
s% 字符串
%% 打印一个百分号

运算符

运算符 说明
+ 从左到右加
- 左侧值减去右侧的值
- 作为一元运算符时改变右侧值的符号
* 将其左侧的值乘以右侧的值

– 前缀模式时对右侧值减一，后缀模式时对左侧值减一
(类型名) 将右侧的值强制转换成圆括号中指定的类型

fgets(arg1, arg2, arg3);

arg1：存储字符串的变量
arg2：字符串长度
arg3：输入模式（获取键盘输入为stdin）

puts()输出字符串

arg：需要打印的字符串

strlen()统计字符串长度

strlen(arg);

arg：待统计的字符串

strcat()拼接字符串

strcat(arg1, arg2);

arg1，arg2：待拼接的两个字符串

arg1变为拼接后的值，arg2不变

strncat()带限制的拼接字符串

strncat(arg1, arg2, arg3);

arg1，arg2：待拼接的两个字符串
arg3：在arg2中选取n个字符拼接到arg1中

strcmp()字符串比较

strcmp(arg1, arg2);

arg1，arg2：两个等待比较的字符串
函数返回值为arg2-arg1 ascii码的差

strncmp()带限制的字符串比较

strncmp(arg1, arg2, arg3);

arg1，arg2：两个等待比较的字符串
arg3：需要比较的前n位字符

strcpy()拷贝字符串

strcpy(arg1, arg2);

arg1：待复制的字符串
arg2：目标字符串

strncpy()拷贝指定位数的字符串

strncpy(arg1, arg2, arg3);

arg1：待复制的字符串
arg2：目标字符串
arg3：可拷贝的最大字符数

第十三章

fopen()函数

用fopen()可以打开一个文件。

fp = fopen(arg1, arg2);

arg1是文件所在目录和文件名
arg2是操作模式，详见下表
fp是返回的文件指针

模式字符串 含义
r 只读模式打开文件
w 覆写模式打开文件
a 追加模式打开文件
r+ 更新模式打开文件（即可读写模式）
w+ 读和写模式打开文件，如文件存在则覆写，如不存在则创建新文件
a+ 读和写模式打开文件，如文件存在则追加写，如不存在则创建新文件，可以读取整个文件，但只能从末尾添加
*b 以二进制模式操作文件
*x 类似于非x模式，但如果文件存在或已经被以独占模式打开，则打开文件失败

getc()和putc()

从fp指向的文件中获取一个字符

ch=getc(fp);

把字符ch放入fpout指向的文件中

putc(ch, fpout);

fclose()

用fclose(fp)来关闭当前打开的文件并解除对其的占用。

其他文件操作函数

函数名 功能
fprintf() 输出指定文件的内容，第一个参数为指向文件的指针
fscanf() 将输入内容写入指定文件中，第一个参数为指向文件的指针
fseek() 将文件内容以数组方式读取
ftell() 返回当前在文件中的位置
fread() 以二进制读取文件
fwrite() 以二进制写文件

第十五章

按位逻辑运算符

按位取反”~”

val = 2;//'b00000010'
val = ~val;//'b11111101'

按位与”&”

int val = 2;//'b00000010'
int val_2 = 7;//'b00000111'
val_2 = val & val_2;//'b00000010'

按位或”|”

int val = 2;//'b00000010'
int val_2 = 7;//'b00000111'
val_2 = val | val_2;//'b00000111'

按位异或”^”

int val = 2;//'b00000010'
int val_2 = 7;//'b00000111'
val_2 = val | val_2;//'b00000101'

掩码

将第二位设为true

#define MASK 2
int flags = 0x96;//‘b10010110'
flags &= MASK;//b'00000010'

应用：打开位

使用”&”运算符开启低位起第四位

#define MASK 8
int flags = 0x96;//‘b10010110'
flags &= MASK;//'b10011110'

应用：关闭位

使用”&”和”~”运算符关闭低位起第四位

#define MASK 8
int flags = 0x9E;//‘b10011110'
flags &= ~MASK;//'b10010110'

应用：切换位

使用”^”运算符切换低位起第三位

#define MASK 4
int flags = 0x96;//‘b10010110'
flags ^= MASK;//'b10010010'

检查位的值

if((flags & MASK) == MASK){
     puts("Wow!);
}

移位运算符

左移：<<

int a = 2;//'b00000010'
a <<= 1;//'b00000100'

右移：>>

int a = 2;//'b00000010'
a >>= 1;//'b00000001'

第十六章

define

作用范围：从’#’到换行符

用于将特定值添加一个代称，不可更改

重定义

替换体中的记号必须相同，且顺序也相同

在define中使用参数

#define MEAN(X,Y) ((X)+(Y)/2)
与函数的区别
  • define只是单纯替换关键词，不会进行计算
  • define不会先计算传入参数的值再操作
  • define在编译之前就把值传给程序

‘##’连接符

#define XNAME(n) variable_##n
int XNAME(2) = 14;//aka 'int variable_2 = 14;'

变参宏

#define PR(...) printf(__VA_ARGS__)
PR("Howdy!");//aka printf("Howdy!");

文件包含

用法示例

#include <stdlib.h>：查找系统目录
#include “glados.h"：查找当前目录
#include /home/pi/fish.h：查找/home/pi目录

使用头文件中的变量

在头文件中加入

extern int status = 0;

即可在主程序中直接使用此变量

其他指令

#undef指令

用于取消#define做出的声明

#define LIMIT 400
#undef LIMIT

条件编译

#ifdef、#else和#endif
#ifdef MAVIS
     #include <horse.h>
     #define STABLES 5
#else
     #include "cow.h"
     #define STABLES 15
#endif

以上代码表示如果定义了MAVIS则执行#else前的所有语句并编译所有C代码，反之则执行#else和#endif之间的所有代码

#ifndef

与#ifdef相反，即当未定义指定标识符时执行#ifndef到#endif之间的语句

#if和#elif语句

如果某个标识符的值被定义为某个值，则执行#if到#elif或#endif之间的语句

#if SYS==1
     #include "ibm.h"
#endif
#error

让预处理器发出一条错误信息，该消息包含指令中的文本

#if __STDC_VERSION__ != 201112L
     #error Not C11!
#endif

以上代码可用来判断C11标准，如果不支持C11标准就会抛出”Not C11!”这个错误消息。

#pragma

可以用此预定义修改一些编译器的参数

#pragma c9x on

上述代码作用是让编译器支持C9X标准

内联函数

作用

减少函数调用时的内存开销

定义内联函数

inline static void eatline(){
     while(getchar() != '\n'){
          continue;
     }
}
