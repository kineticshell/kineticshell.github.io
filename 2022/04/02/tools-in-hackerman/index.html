盘点《黑客军团》第一季中出现的那些真实存在的工具软件

Linux 网络安全 黑客军团 渗透工具
杂谈
发布日期:   2022-04-02

没什么用处的前言

《黑客军团》（英文名：Hackerman Mr.Robot）算是在一众好莱坞版特效黑客里考据最准确的剧了，追这个剧的时候也见到了不少熟悉的面孔（指软件），所以打算边看剧同时边整理整理这个剧里出现的那些有意思的工具，就当是追剧笔记了（笑）。

S01E01

Tor

第一幕，我们的Elliot威慑餐馆老板

洋葱网络，就是Tor，The Onion Router的缩写，是一种开源的匿名通信软件。其通过众多的中继服务器和加密来实现匿名性，但正如Eillot所说，Tor不能加密出口节点到目标服务器的流量，控制出口网络节点的人就能完全掌握所有通信内容，因此Eillot就凭借这个弱点搜集到了餐馆老板的全部通信数据。

GNOME vs KDE Plasma



两个Linux系统中比较知名的图形化桌面环境（Desktop Environment），GNOME比较适合那些想要上手即用，追求高效的用户，KDE PLasma的插件和自定义参数很多，适合那些比较喜欢折腾的用户。r/Unixporn老哥表示真折腾还得i3wm/awesome和bspwm

不过这里有一点E公司小哥说的不太准确，KDE是KDE自由软件社群的名字，而他们开发的桌面环境叫KDE Plasma，所以应该称其为KDE Plasma。

RUDY攻击

全称是aRe yoU Dead Yet?是一种DDoS（拒绝服务攻击）工具的名称，与一般的GET型DDoS攻击不同，它可以通过构建少量但持续的异常POST请求来对目标服务器进行瘫痪，通常针对那些带有表单的网站。这种攻击会在POST请求得到接受后不断发送非常少量的数据，长时间不终止传输会话，使得目标服务器的带宽被持续占用从而达到攻击目的。阅读更多…

set


用set命令修改Cisco服务器的waneth0口的IP地址，没什么好说的。

more

more命令：用于在终端将可能很大的文件以文本形式输出时的命令。不过我一般用cat。

ps和grep

ps命令：用于显示Linux中正在运行的服务和进程。

grep命令：用于在输入中查找指定关键词并返回高亮结果，支持正则表达式。

pipe传递符命令：即|，用于将其前面命令的输出作为输入传递给下一个命令。
此处Elliot命令的意思是查找所有以root身份运行的进程和服务。

chmod

chmod命令：用于修改文件的权限，这里Elliot的命令是使用八进制权限模式将文件设置为只有所有者才能读写的权限。

剧组自创命令

astsu这个没有现实原型，推测可能是Evil Corp内部使用的CLI服务器管理程序。

elpscrk这个也没有现实原型，但看起来应该就是一个密码爆破工具，有点像Hydra和Hashcat。

社会工程学

这段是我认为第一集里最精彩的部分。Elliot发现这个目标使用了假名，且几乎查不到与之相关联的社交网络账户，于是他通过心理咨询师与目标约会的机会骗到了目标的手机，发现了其中的Bank of E，得知目标拥有一个Bank of E的账户。随后他就使用伪造的号码假扮Bank of E工作人员以帐号被入侵为由询问目标的敏感信息和密保问题，最后成功得到了目标的正确密码，简直是米特尼克《欺骗的艺术》的完美示范。

S01E02

Shellshock

Shellshock，正如字幕组科普，是在Unix-like系统中普遍采用的shell程序“bash”中的一个参数解析漏洞，攻击者可以构造不合法的请求来实现远程执行任意代码。代号CVE-2014-6271，感兴趣的朋友可以下载这个github上的docker镜像来自己体验一番。

wget、非法UA与RCE

Elliot使用wget指令对上述漏洞的利用。代码如下（网址部分已省略）

wget -U "() { test;}; echo \"Content-type: text/plain"; echo; echo; /bin/cat /etc/passwd" http://ur.mom

wget是GNU里的一个下载器，参数-U是用于构建自定义的User-Agent。这里Elliot输入的这些奇怪的字符就是为了利用这个漏洞，使其无法被邮件服务器作为文本正常解析，从而让后面的/bin/cat /etc/passwd命令得到调用。不过根据github上的poc命令来看这里Elliot似乎在test和分号之间少打了个冒号。随后Elliot成功把目标服务器上的/etc/passwd作为status文件存到了本地，并使用cat指令进行了查看，得到了目标服务器上的账户信息。关于两步验证这段也是个有意思的吐槽，是的，两步验证也许在日常使用时有些麻烦，但它在数据库泄露事件发生时真的可以救你账户一命。

John


Elliot用John the ripper jumbo工具搭配他的字典来跑hash从而试图得到Tyrell的密码。
不过这段有两个槽点，可能是剧组考虑不周。
第一点：Elliot的命令是

root@elliot:~#./john /etc/status

可是这个status并不是他刚刚用wget指令下载下来的status文件，结合终端的~来看Elliot下载的status文件应该是保存在/root目录才对。而且即使那个status是Elliot提前下载好的hash密钥文件他把这文件放到自己系统的/etc目录下也很奇怪。
第二点：Linux系统中/etc/passwd并不包含真正的密码hash值，其中储存的是这台服务器的用户列表信息。真正的密码hash文件则存放在/etc/shadow中。这里Elliot只下载了目标服务器的passwd文件并命名为status却并没有下载shadow文件以及用john试图破解的文件也叫status我怀疑也是剧组的疏漏，错把passwd当成储存密钥的文件了。

物理骇入（撬锁）


虽然“黑客攻击（物理）”通常被看作一个玩笑，但开锁技能在一些特定场合还是非常有用的。你可以在这里学到如何使用回形针撬锁。

后渗透操作之摄像头偷窥

渣男男三被一张“音乐”CD狠狠坑了一把。攻击者查看的画面正是他通过CD感染渣男电脑后开启的摄像头流式传输页面。现实中的metasploit附带的meterpreter服务端也有同样的功能，成功在宿主机执行后，它就可以在没有任何提示下静悄悄地开启摄像头并通过网页来进行流式传输，受害者的一切都会暴露在攻击者眼中。所以普通人为了保险起见，买个摄像头屏蔽盖还是非常有必要的。当然也不要随便把捡来的U盘/CD插入你的电脑中。

S01E03

暴力破解

蛮力攻击（英语：Brute-force attack），是利用字典或设定好的程序/脚本不断对加密数据尝试解密。暴力破解看似十分低效，但由于大部分人安全意识薄弱，密码设置得十分简单，8位以内纯数字字母组合密码在hashcat调用GPU算力的情况下不到一小时即可破解完毕。如果再加上前文提到的Eillot使用的那种社会工程学，暴力破解工具将成为黑客们手中的一把利器。暴力破解工具可参考前文提到的hashcat、hydra、john等。此外还有metasploit自带的mimikatz和kiwi。

Symantec Endpoint Protection

Elliot受伤住院，特意选择了自己黑入过的这家医院来修改自己的血检报告来掩盖自己磕药的事实。截图里的杀软就是Symantec开发的用于企业安全的赛门铁克端点防护。杀毒能力一流，但是遇到无法联网更新病毒库的医院内网环境。。。结果显而易见。

附上一段Elliot的吐槽：

在国内的话大概就是Windows XP + IE + Flash Player组合拳了吧（笑）

钓鱼攻击

建立一个临时的伪造网站，随后将伪造好的十分具有诱惑力的邮件发送到目标的邮箱中，等待目标上钩。这就是钓鱼攻击。不能理解？还记得诸如“免费送Q币”、“免费送QQ黄钻会员”、“登陆qq邮箱以查看高清通知图”么？很蠢，但就是有大聪明上钩。（笑）

后门攻击

软件后门指绕过软件的安全性控制，从比较隐秘的通道获取对程序或系统访问权的黑客方法。（摘自维基百科）
后门通常是开发商或程序员在编程时用于调试的隐藏指令或账户，通常拥有非常高的权限。一旦这些后门账户/指令在最终发行版中忘记去除，将会造成非常严重的后果。比较著名的就是海康威视的Magic String Backdoor，只需在IP摄像头的地址栏加上一个

snapshot?auth=YWRtaW46MTEK

即可直接得到该摄像机的实时画面，随后该漏洞被进一步利用，甚至可以直接得到密码和管理权限。造成这一严重事故的直接原因就是海康威视的开发人员在V5.4.0build 160921版本的固件中忘记删除了开发者测试后门所导致的。

FramaRoot和SuperSU

在成功get到对方服务器后门权限以后，Tyrell在对方的手机上安装了这么一款叫RooterFrame的app，功能似乎是一键Root。经查证这个工具虽然不存在，但它的原型是FramaRoot，类似于国外版的KingRoot。利用了Boromir Exploit、Gandalf exploit和Pippin Exploit等漏洞利用脚本来实现一键root。不过现在主要使用Magisk+TWRP的root方式，以上这些属于是Android2.3-4.4时代的眼泪了。

SuperSU，Android2.3-4.4时代最出名的Root管理器，没什么好说的。我对它印象最深的就是它可以实时修改自己的APP图标。

抓包与Headers

这段应该是抓包数据，这一行数据分别对应来源地址、时间戳、方法、协议、状态、对应端口和User-Agent

ipv6

S01E04

树莓派

树莓派是一种起初被用作教育目的的开源卡片型电脑，采用arm架构的CPU。后来因为其不错的性能，丰富的拓展，完善的接口和可运行Linux发行版使得它在极客和DIY爱好者们中受到极大欢迎，许多人用它来制作各种智能家居的主控系统，智能音箱，“魔镜”等实用或奇妙的小工具。当然也包括像Elliot这样的信息安全人员用它来当作便携式信息安全测试工作站。

虚拟专用网络

俗称代理或VPN，指用户通过一台远程服务器作为中转来代替自己的电脑发出请求。因为它高度的匿名性，因此许多人用它来避免广告跟踪与内容审查。此外，它还可以作为一种访问未公开局域网的手段：用户将一台同时可以访问内网和互联网的服务器上部署VPN服务，用户即可通过此服务器来获取内网访问权限。这也正是Elliot打算做的。

FTP


FTP，全称File Transfer Protocol，是一种专用于文件传输的网络协议。优点是简单便捷，缺点是明文传输，无加密，极易遭到中间人攻击导致传输内容被泄露。sftp这种基于ssh的文件传输协议正在逐步取代ftp。

Odb2设备与CAN Bus


这段我实在不熟，根据一篇reddit帖子的分析其应该是利用专用的嗅探器嗅探了汽车内的控制器局域网，抓到了开锁时的握手包，对其解密后成功打开了汽车的锁。（以后如果对此领域有涉足将会补充）

守护进程

守护进程，英文名是daemon,源于守护天使的英文。是一种在后台持续运行的服务程序，名称通常以字母“d”结尾，通常用于对某个进程保活。字幕组翻译成后台进程其实有些不太准确，守护进程是后台进程的一种，定义范围更加狭窄。

tar

tar是一种Unix-like系统自带的纯命令行打包工具，与winrar类似。唯一的区别在于其打包后的后缀名为*.tgz或*.tar.gz。tar的用法可见此：tar man page

FileZilla

FileZilla是一款开源的GUI文件传输客户端，支持ftp，sftp以及ipfs协议。我个人就经常用它来在我的电脑和手机之间互传文件，所以当这个镜头出现时我马上认出了这个工具。

Arch系Linux发行版可以直接通过

sudo pacman -Syu filezilla

安装。
